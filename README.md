# Jail-website  SQL ONLY, WITHOUT FRON END

This project was made using PHP and SQL

# Project overview
 - Website is a personal representation of website for a Jail
 - Personal trainig with Sending and Receiving the SQL DB info 
 - Usage of One to Many and Many to Many relationships

# Running with XAMPP
 - Save the folder in folder: C:\xampp\htdocs
 - Start the XAMPP and Apache
 - Use http://localhost/Jail-website/Main_page.php  link to see the website
 - Use http://localhost/phpmyadmin/  to se the Database
